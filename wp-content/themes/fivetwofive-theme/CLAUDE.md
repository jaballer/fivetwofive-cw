## Project Overview

FiveTwoFive Theme is a custom WordPress parent theme (v0.9.2) for FiveTwoFive Creative projects. It uses ACF Pro for modular page building, WordPress Customizer for global styling, and a Gulp-based asset pipeline.

**Requires:** ACF Pro, Bootstrap 5.2.3, jQuery (WordPress bundled)

## Development Commands

```bash
npm start          # Dev mode: compile SASS/JS + BrowserSync
npm run build      # Production: compile and minify all assets
npm run watch      # Watch without BrowserSync
gulp styles        # Compile SASS only
gulp scripts       # Lint + transpile + minify JS
gulp lint          # ESLint only
gulp imageminify   # Optimize images
```

**BrowserSync setup:** Copy `gulpfile-config-sample.js` → `gulpfile-config.js` and set proxy URL to your local WordPress site.

**Never modify `assets/dist/`** — always edit sources in `assets/src/` and run build.

## Architecture

### Entry Points

- `functions.php` — orchestrates all includes in order
- `page-templates/template-module.php` — the ACF flexible content page builder template
- `template-parts/modules/module.php` — dispatches to individual module templates

### PHP Class Files (`classes/`)

- `class-fivetwofive-theme-customize.php` — Customizer panel/section/control registration
- `class-fivetwofive-theme-css-php.php` — Generates dynamic CSS from theme mods (injected inline)
- `class-fivetwofive-theme-svg-icons.php` — SVG icon library (53KB, 50+ social icons)
- `class-fivetwofive-theme-customize-select2-control.php` — Select2 UI for font picker

### SASS Architecture (ITCSS pattern)

```
assets/src/sass/
├── abstracts/   variables (_colors, _typography, _structure, _columns), functions, mixins
├── generic/     normalize, box-sizing
├── base/        elements (body, buttons, links...), typography
├── components/  header, footer, navigation, card, pagination, share-buttons, tooltips, hamburgers (35+ types)
├── modules/     one partial per ACF module
├── layouts/     site, page-layout, sidebar variants
├── templates/   per-post-type styles (archive, single-event, single-post, single-resource, 404)
└── utilities/   accessibility, alignments, backgrounds
```

`main.scss` imports everything. Individual post-type stylesheets (e.g., `single-event.scss`) are compiled separately and conditionally enqueued.

### JavaScript Architecture

Core scripts in `assets/src/js/`:
- `navigation.js` — mobile nav
- `view-toggle.js` — grid/list view with `localStorage` persistence
- `single-post.js` — social sharing tooltips (Popper.js)
- `template-modules.js` — ScrollReveal integration for module animations

Module scripts in `assets/src/js/modules/`:
- `module-accordion.js` — jQuery UI accordion
- `module-announcement.js` — dismissible banners
- `module-resource.js` — AJAX filtering + pagination via REST API
- `module-works.js` — portfolio filtering + ScrollReveal
- `module-testimonials-carousel.js` — Swiper carousel

JS is transpiled with Babel and minified. Use IIFE or ES6 class patterns (see `module-resource.js` for class example).

## Key Subsystems

### Asset Enqueuing (`inc/public/assets.php`)

Assets are loaded conditionally:
1. **Global** — main CSS, Google Fonts, navigation
2. **Post-type specific** — `single-event.css`, `single-resource.css`, `single-music.css`
3. **Blog/archive** — `view-toggle.js`
4. **Single posts** — `single-post.js` + Popper.js
5. **Module template only** — Fancybox, Swiper, ScrollReveal, all module scripts

Scripts in the `$defer_scripts` array are automatically given `defer` attribute via `fivetwofive_theme_defer_scripts()`.

Google Fonts URL is dynamically generated by `fivetwofive_theme_generate_google_fonts_url()` — handles multiple families, italic/regular variants, `display=swap`.

ACF fields `ftf_custom_styles` / `ftf_custom_scripts` inject page-level inline styles/scripts. `ftf_module_styles` / `ftf_module_scripts` inject module-level overrides.

### WordPress Customizer (`inc/admin/customizer.php`)

Theme mods are accessed via `fivetwofive_theme_mods()`, which merges saved values with defaults defined in `fivetwofive_theme_default_theme_mods()`.

**Color controls** (`colors_*` prefix): body background/text/heading/links, button states, header, primary navigation, footer.

**Typography controls** (`typography_*` prefix): body font + heading font — each with name, variants, category, weight. Integrates with Google Fonts (800+ fonts via Select2).

Override defaults in a child theme:
```php
add_filter( 'fivetwofive_theme_default_theme_mods', function( $mods ) {
    $mods['colors_body_text'] = '#333333';
    return $mods;
});
```

### Admin: Content Editor Manager (`inc/admin/content-editor-manager.php`)

Automatically hides the WordPress editor for pages using `template-module.php`. Uses dual hooks (`admin_init` + `current_screen`) with multiple template detection methods for reliability.

### SVG Icons

```php
echo fivetwofive_theme_get_icon_svg( 'ui', 'play', 100 );   // UI icons
echo fivetwofive_theme_get_icon_svg( 'social', 'youtube', 40 ); // Social icons
```

Shortcode: `[fivetwofive_icon group=social icon=youtube size=40]`

Extend via filter:
```php
add_filter( 'fivetwofive_theme_svg_icons_ui', function( $arr ) {
    $arr['checkmark'] = '<svg viewBox="0 0 32 32">...</svg>';
    return $arr;
});
```

## Modules (ACF Flexible Content)

13 modules in `template-parts/modules/`. The module loader (`module.php`) dispatches based on ACF layout name. All modules are loaded on pages using `template-module.php`.

| Module | Description |
|---|---|
| module-accordion | Collapsible sections |
| module-announcement | Dismissible banners |
| module-code | Embed code/shortcodes |
| module-content-and-media | Content + image/video |
| module-cta | Call-to-action |
| module-events | Event listing with filters |
| module-gallery | Fancybox lightbox gallery |
| module-hero | Hero with background options |
| module-multi-column | 2–5 column responsive grid |
| module-resources | AJAX-filtered resource list |
| module-testimonials-carousel | Swiper carousel |
| module-works | Portfolio with filtering |

Child themes can override any module by placing a matching file in their own `template-parts/modules/`.

ACF field group JSON is version-controlled in `acf-json/`. The main modules group (`group_5c9b8ee97b445.json`) is 337KB.

## Key Filters & Hooks

```php
// Layout control
add_filter( 'fivetwofive_theme_enable_sidebar', fn($e) => true );     // enable sidebar
add_filter( 'fivetwofive_theme_is_contained', fn($c) => false );      // full-width

// Hook after main content (sidebar injection)
do_action( 'fivetwofive_after_content' );
```

## Child Theme Pattern

This is a parent theme. Child themes can:
- Override template parts including individual modules
- Add filters to customizer defaults, layout, and icons
- Enqueue additional styles/scripts
- Register additional ACF field groups (saved to child theme's `acf-json/`)

## REST API

Custom endpoints in `inc/public/rest-api.php` for resource filtering and module data. The REST base URL is injected into JavaScript as `FiveTwoFive.restBase` via `wp_add_inline_script`.

## Custom Plugins

All custom plugins live in `wp-content/plugins/` with a `fivetwofive-` prefix. They extend the theme with post types, taxonomies, and utilities, and many hook directly into theme filters.

### Post Type Plugins

**fivetwofive-events-post-type**
- Registers `ftf_event` CPT (REST enabled, no archive)
- Hooks into `fivetwofive_theme_is_contained` (single events → full-width) and `fivetwofive_theme_enable_sidebar` (archives → no sidebar)
- ACF fields (registered via theme's `acf-json/`): `ftf_event_type`, `ftf_event_start_date`, `ftf_event_end_date`, `ftf_event_start_time`
- Outputs event metadata via `fivetwofive_theme_after_post_meta` action

**fivetwofive-resource-post-type**
- Registers `ftf_resource` CPT (REST base: `ftf-resources`) + `ftf_creative` CPT (admin-only, for team/creative directory)
- Taxonomies: `ftf_resource_category` (hierarchical) and `ftf_resource_tag` (flat), both REST-enabled
- Image size: `ftf-resource-thumb` at 415×245
- Extends REST responses with formatted category/tag arrays; hooks into `fivetwofive_rest_api_formatted_date`
- Hooks into `fivetwofive_theme_is_contained` (full-width singles)

**fivetwofive-work-post-type**
- Registers `ftf_work` CPT (REST enabled, no archive)
- Taxonomy: `ftf_work_category` (hierarchical, REST enabled)
- Image size: `fivetwofive-work-thumbnail` at 600×450
- Includes Gutenberg block assets: `blocks/variations.js` (editor) + `css/style-work-archive-cards.css` (front + editor)

**fivetwofive-music-post-type**
- Registers `music` CPT (archive at `/music/`)
- Taxonomies: `music_genre` and `music_type` (both hierarchical)
- Hooks into `fivetwofive_theme_is_contained` (full-width singles)

**fivetwofive-daily-quotes**
- Registers `daily_quote` CPT (archive at `/quotes/`)
- Shortcode `[daily_quote]` — outputs latest published quote

**fivetwofive-lp-post-type**
- Registers `fivetwofive-lp` CPT (no archive, supports categories/tags)
- Removes the WordPress editor on `admin_init` when the page uses `template-module.php`

### Utility Plugins

**fivetwofive-cta**
- Global CTA block managed via an admin settings page (Settings → Call To Action)
- Shortcode `[fivetwofive_cta]` renders the configured CTA; attributes can override global settings
- Settings: title, message, button text/URL/target, text alignment, background image/color, font colors
- OOP architecture with Composer autoloader (`FiveTwoFive\FiveTwoFive_CTA` namespace)
- Admin assets: color picker (WordPress Iris) + jQuery UI tabs + media uploader

**fivetwofive-template-column-display**
- Admin-only utility — adds a sortable "Template" column to the Pages list screen
- No frontend output, no assets

### Theme–Plugin Integration Pattern

Plugins use theme filters rather than duplicating layout logic:
- `fivetwofive_theme_is_contained` — most CPT plugins set single posts to full-width
- `fivetwofive_theme_enable_sidebar` — archive pages opt out of the sidebar
- `fivetwofive_theme_after_post_meta` — CPT plugins inject their custom meta below the post header
